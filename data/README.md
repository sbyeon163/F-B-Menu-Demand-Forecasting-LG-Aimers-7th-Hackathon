# 📂 Data Directory

본 프로젝트에서 사용된 데이터는 **LG AI Research**에서 제공한 해커톤용 데이터셋으로, 저작권 및 보안 규정 준수를 위해 원본 파일은 포함하지 않았습니다.

### 📊 데이터셋 정보 (Dataset Information)
- **파일명**: `train.csv`, `test.csv`
- **주요 내용**: 1.5년 기간의 리조트 내 식음업장(F&B) 메뉴별 일일 판매 데이터
- **특이사항**: 
    - 매출 0인 날이 50% 이상 발생하는 고희소성(High-sparsity) 데이터

### ⚙️ 데이터 구성 가이드 (Setup Guide)
분석을 재현하려면 아래의 단계를 따라주세요.
1. Dacon에서 식음업장 메뉴 수요 예측 AI 온라인 해커톤을 검색 후 제공된 `train.csv`와 `test.csv`를 다운로드합니다.
2. 해당 파일들을 이 `data/` 폴더 내에 배치합니다.
3. `scripts/` 폴더 내의 전처리 및 모델링 코드를 실행합니다.
